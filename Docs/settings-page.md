# Settings Page

The Settings page is where the user manages app-wide preferences and maintenance actions.

## What the page shows

The screen groups settings into sections:

1. Vehicle management.
2. Basic preferences such as currency and distance unit.
3. Data actions such as export and clear-all-data.

It also shows the app version at the bottom.

## How it works

The vehicle management entry navigates to the Vehicles page.

The clear-data action opens a confirmation dialog before deleting all stored vehicles, trips, and fuel logs. It also clears the active vehicle selection.

## Purpose

This page is the control center for configuration and data maintenance. It is where the user goes to manage the app state without interacting with the tracking pages directly.
